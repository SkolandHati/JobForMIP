## JobForMIP

Установка зависимостей

```
# npm
npm install

```

## Запуск в режиме разработки

Start the development server on `http://localhost:3000`:

```
# npm
npm run dev
```

## Сборка

Сборка продукта:

```
# npm
npm run build
```

Предварительный просмотр сборки:

```
# npm
npm run preview
```
